## FLUXO DE TRABALHO GIT LOCAL

1. git checkout -b "nova branch"
2. atualiza ou cria arquivos
3. git add .
4. git status
5. git commit "mensagem"
6. git checkout main
7. git merge nova branch

## Fluxo de trabalho GitHub <> local (projeto proprio ou da empresa)

1. git clone <url do projeto>
2. git checkout -b <nova branch>
3. alteracoes do arquivos
4. git status
5. git add <arquivos>
6. git status
7. git commit -m "nova mensagem"
8. git push origin <nova branch>
9. abrir pull request no github para main
10. excluir <nova branch> origin
11. git checkout main
12. git branch -d <nova branch>
