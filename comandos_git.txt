git init .              - Cria um repo git na pasta atual
git status 		- Mostra o status do trabalho do arquivo
git add nome_arquivo - Adiciona arquivo p ser commitado
git commit -m "descricao do commit" - Faz commit doq estava p ser commitado
git log 		- Mostra todo o historico de commit

## FLUXO DE TRABALHO

1. cria diretorio do projeto
2. iniciar o git do diretorio criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add "nome do arquivo"
6. git status
7. git commit -m "mensagem estranha"
8. git status
9. volta para o passo 3
